Sobre a Loja

Loja criada com HTML e CSS apenas para a demonstração dos conhecimentos das TAGS semânticas
aprendidas durante a disciplina de Introdução a Prgramação WEB do período 2024.1.
Feito pelo aluno João Vitor Pê Nascimento Lins, as páginas HTML criadas possuem as TAGS semânticas ensinadas e
requiridas na atividade, como por exemplo as TAGS: "section, article, nav, footer, main, header, etc".
Há também a utilização de um CSS básico para o embelezamento e melhor organização das páginas.
